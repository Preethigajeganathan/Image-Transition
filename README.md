# 🖼️ Simple Image Transition

A simple and responsive Image Transition Application built using **HTML, CSS, and JavaScript** that allows users to compare two versions of an image using an interactive slider effect.

## 🚀 Live Demo

🔗 https://preethigajeganathan.github.io/Image-Transition/

## 🚀 Features

* Interactive image transition effect
* Compare original and background-removed images
* Smooth slider movement
* Mouse support for desktop devices
* Touch support for mobile and tablets
* Responsive design
* Lightweight and easy to customize

## ▶️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/Preethigajeganathan/Image-Transition.git
```

### 2. Navigate to the Project Folder

```bash
cd Image-Transition
```

### 3. Open the Application

Simply open `index.html` in your browser.

## 🎮 How to Use

1. Open the application in a browser.
2. Move your mouse over the image on desktop devices.
3. Drag your finger across the image on mobile or tablet devices.
4. Compare the original image with the background-removed version using the slider.
